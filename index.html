常见问题

条码是否全国通用，各地都可以使用？▾
是的，通过正规渠道办理的商品条码（厂商识别代码）由中国物品编码中心统一管理，全国各地、各大商超、电商平台均可正常使用，同时也是国际通用标准（EAN-13）。
办理的条码是否可以在京东/天猫等平台使用？▾
可以。京东、天猫、拼多多等主流电商平台在新品发布时均需上传商品条码，我们办理的条码完全符合各大平台要求，可直接用于商品录入与发布。
条码有效期是多久？到期了怎么办？▾
条码有效期为2年，到期前需办理续展，否则条码将失效，影响产品正常流通。我们提供续展提醒服务，建议到期前3个月办理续展。
一个厂商识别代码可以生成多少个条码？▾
一个厂商识别代码可生成1000个条形码，可用于1000个不同商品/规格。如需可下载打印的电子矢量胶片，官方售价32元/个（部分地区60元/个），我们免费赠送！如产品线较多，可申请增号服务。

条形码生成服务

69码与店内码的区别，一次看懂

69开头条码（EAN-13）

🏷️ 标准商品条码

由厂商识别代码 + 商品项目代码 + 校验码组成，是产品包装印刷和商超流通的法定标准条码。

  • ✓ 需先注册厂商识别代码
  • ✓ 可在商超/电商正常使用
  • ✓ 全国/全球范围内可识别
  • ✓ 提供胶片制作，支持印刷
  • ✓ 扫码可查出产品厂商信息
店内码（内部条码）

📦 内部管理条码

企业自定义的内部管理条码，不需要注册厂商代码，仅用于企业内部系统管理，无法在外部平台识别。

  • ✓ 无需注册，快速生成
  • ✓ 仅用于店内/仓储管理
  • ✓ 含条码图片 + 编码
  • ✗ 不可在商超流通使用
  • ✗ 不可用于电商平台入驻
  • ✗ 外部扫码不可识别
